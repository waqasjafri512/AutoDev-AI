# AutoDev AI Frontend

A premium React dashboard built with **Tailwind CSS v4** and **Lucide Icons**.

## 🎨 Tech Stack

-   **Framework**: [React](https://reactjs.org/) (Vite)
-   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
-   **Icons**: [Lucide React](https://lucide.dev/)
-   **Routing**: [React Router v6](https://reactrouter.com/)
-   **State**: React Context API (Auth, Theme, Socket)

## 🌙 Dark Mode
The application supports a global dark mode toggle. The theme is persisted in local storage and defaults to system settings.

## ⚙️ Environment Variables

```env
VITE_API_URL=http://localhost:3000
VITE_STRIPE_PUBLISHABLE_KEY=
VITE_GITHUB_CLIENT_ID=
```

## 🚀 Run Locally
```bash
npm install
npm run dev
```
